# 会帮手 用户操作说明书

## 一、软件简介

会帮手（MeetTimer）是一款专为 Windows 桌面打造的会议计时工具。软件采用 **PySide6 + SQLite** 技术实现，帮助用户高效管理会议时间，提高会议效率。核心功能包括：会议计划管理（支持 Excel 批量导入）、阶段计时（汇报/讨论）、悬浮计时窗口、语音提示、历史记录统计与导出。

## 二、界面概览

### 主界面布局

1. **顶部导航栏**：包含 Logo、导航标签（会议安排 / 当前会议 / 历史记录）、设置按钮和悬浮窗口按钮
2. **主内容区**：使用堆叠页面切换不同功能面板
3. **当前会议页**：左侧为计时器控制区，右侧为议题计划状态表

### 导航标签

| 标签 | 说明 |
|------|------|
| 会议安排 | 创建和管理会议议程，支持 Excel 导入和模板保存 |
| 当前会议 | 会议进行中的计时界面（仅在会议启动后显示） |
| 历史记录 | 历史会议记录的查看、搜索、导出和删除 |

## 三、功能说明

### 3.1 会议安排

在会议安排页面，您可以：

#### 创建会议计划
1. 在「会议名称」输入框中输入会议名称（可为空，启动时自动生成时间戳名称）
2. 点击「添加议题」按钮添加新议题（使用设置中的默认汇报/讨论时长）
3. 双击单元格可直接编辑议题名称、汇报时间、讨论时间
4. 支持拖拽调整议题顺序
5. 点击表头复选框可全选/取消全选议题
6. 点击「删除选中」删除已勾选的议题

#### Excel 导入会议议程
- 点击会议名称右侧的「导入会议计划」链接
- 弹出导入对话框：
  - **下载模板**：下载标准 Excel 模板（包含「议题」「汇报时间（分钟）」「讨论时间（分钟）」三列）
  - **导入文件**：选择 Excel 文件导入，自动解析并填充议程
  - **取消**：关闭对话框
- 导入时如当前已有数据，会提示确认覆盖
- 导入后 Excel 文件名（不含扩展名）自动作为会议名称

#### 保存与加载会议计划
- **保存会议计划**：将当前议题列表保存为模板（会议名称为空时自动使用时间戳）
- **加载会议计划**：从已保存的模板列表中选择加载，自动填充会议名称和议题
- **管理会议计划**：
  - 编辑计划：加载模板到编辑区进行修改
  - 修改名称：重命名已保存的模板
  - 删除选中：删除不再需要的模板

#### 开始会议
- 点击「开始会议」按钮启动计时器
- 自动切换到「当前会议」页面，同时开启悬浮窗口

### 3.2 当前会议

当前会议页面是会议进行时的主要操作界面，左右分栏布局：

#### 左侧：计时器控制区
- **会议名称**：显示当前会议名称
- **议题名称**：显示当前议题名称
- **阶段显示**：汇报时间 / 讨论时间
- **实时计时**：大号时钟显示倒计时或超时时长
- **统计信息**：计划时间、当前状态（汇报中/讨论中/暂停/超时）

#### 控制按钮

| 按钮 | 功能 |
|------|------|
| 进入讨论 | 结束当前汇报阶段，切换到讨论阶段 |
| 暂停 / 继续 | 暂停/恢复当前阶段计时 |
| 重置 | 确认后将当前阶段计时归零并重新开始 |
| 下一议题 | 跳过当前阶段进入下一议题（讨论阶段直接跳议题，汇报阶段先跳讨论） |
| + 临时议题 | 在会议进行中临时添加新议题 |
| 结束会议 | 确认后结束会议，自动显示统计结果 |

#### 右侧：议题计划状态表
- 显示所有议题的序号、名称、汇报时间状态、讨论时间状态
- 状态标记：○ 待开始、⏱ 进行中、✓ 已完成
- 进行中的议题以蓝色高亮显示，已完成议题灰色显示
- 支持上移/下移调整未开始议题的顺序

#### 临时添加议题
- 输入议题名称、汇报时间、讨论时间（默认值来自设置）
- 选择插入位置：当前议题之后 / 末位
- 自动创建阶段记录并刷新议题表

### 3.3 悬浮窗口

悬浮窗口为圆形计时器，始终显示在屏幕最前端：

#### 显示效果
- 圆形渐变背景，扇形饼图显示剩余时间比例
- 蓝色：正常计时 | 橙色：接近超时（根据设置中的提前时长） | 红色：超时
- 超时时叠加红色闪烁脉冲效果
- 暂停时显示"暂停"文字（黄色背景闪烁）
- 大号居中时间显示，顶部标注"剩余时间/超时"，底部标注"汇报中/讨论中"

#### 操作方式
- **拖拽移动**：鼠标左键拖拽可移动位置（位置自动记忆）
- **双击关闭**：双击悬浮窗口可关闭（同时取消导航栏按钮选中状态）
- **右键菜单**：暂停/继续、重置、进入讨论、下一议题、临时议题、结束会议

#### 尺寸与设置
- 小（100px）/ 中（120px）/ 大（150px），在设置中配置或快捷键切换
- 透明度 10%~100% 可调

### 3.4 历史记录

- 列表显示所有已完成的会议，包含复选框、会议名称、日期、议题数
- **搜索**：输入会议名称或日期关键词实时过滤
- **复选框**：表头复选框可全选/取消全选
- **详情预览**：选中会议后在下方面板预览议题明细（日期、计划、实际、超时）
- **双击**：双击会议记录弹出详细统计对话框
- **导出**：勾选会议后批量导出为 Excel 文件（支持覆盖确认）
- **删除选中**：确认后批量删除勾选的会议记录

### 3.5 会议统计

- 会议结束后或双击历史记录时弹出
- 表格展示每个议题的汇报/讨论阶段明细：议题名称、阶段、计划时长、实际用时、超时时长、占比
- 底部合计行汇总全部数据
- 支持导出 Excel（带格式化的表头和汇总行）
- 导出成功后可点击"打开文件路径"定位文件

### 3.6 设置

设置窗口包含两个标签页：

#### 基础配置
- **议题默认时间**：新建议题时的默认汇报时间和讨论时间（分钟）
- **悬浮窗口透明度**：滑块调节 10%~100%
- **悬浮窗口大小**：小 / 中 / 大

#### 提示设置
- **提示音设置**：支持刷新声音列表（自动加载 Sound 目录下的 WAV 文件）
- 三种场景分别配置：
  - **剩余时间提醒**：选择提示音 + 提前多少分钟提醒
  - **时间到**：选择提示音（阶段时间用完时播放）
  - **超时提醒**：选择提示音 + 每隔多少分钟重复提醒
- 每种提示音可选择预设 WAV 文件、语音播报（TTS）或无声
- **试听**按钮可预览选中的提示音效果
- 超时语音播报会语音提示"汇报已延时X分钟"或"讨论已延时X分钟"

### 3.7 会议恢复

软件启动时会自动检测是否有未完成的会议（如上次异常退出），弹出对话框询问是否恢复：
- 确定恢复：加载上次的会议状态、当前议题和阶段
- 自动恢复计时器状态（倒计时/暂停/超时）
- 超时状态下恢复会抑制初始闪烁效果

## 四、快捷键说明

| 快捷键 | 功能说明 |
|--------|----------|
| `Ctrl + F9` | 切换悬浮窗口为小尺寸并显示 |
| `Ctrl + F10` | 切换悬浮窗口为中尺寸并显示 |
| `Ctrl + F11` | 切换悬浮窗口为大尺寸并显示 |
| `Ctrl + F12` | 显示/隐藏悬浮窗口 |

## 五、使用流程

### 5.1 创建并启动会议（手动添加议题）

1. 进入「会议安排」页面
2. 输入会议名称
3. 点击「添加议题」逐个添加议题
4. 双击单元格编辑议题名称和时间
5. 点击「开始会议」启动计时

### 5.2 创建并启动会议（Excel 导入）

1. 点击「导入会议计划」链接
2. 选择「下载模板」获取标准 Excel 模板
3. 在 Excel 中填写议题、汇报时间、讨论时间
4. 再次点击「导入会议计划」→「导入文件」
5. 确认覆盖提示后自动填充
6. 点击「开始会议」

### 5.3 会议控制流程

- 每个议题分为**汇报时间** → **讨论时间**两个阶段
- 汇报阶段点击「进入讨论」切换到讨论阶段
- 讨论阶段点击「下一议题」进入下一议题的汇报阶段
- 可随时暂停/继续、重置当前阶段
- 可随时添加临时议题

### 5.4 查看与分析

1. 会议结束后自动弹出统计对话框
2. 在「历史记录」页面可查看/搜索所有历史会议
3. 勾选会议后可批量导出 Excel 或删除记录
4. 双击会议记录查看详细统计

## 六、数据存储

- 数据库使用 SQLite（WAL 模式），位于 `%LOCALAPPDATA%\会帮手\gac_timer.db`
- 所有会议记录、议题、计时阶段数据均持久化保存
- 设置项（默认时间、显示参数、提示音配置）以 JSON 格式存储在 `app_settings` 表中
- 自定义提示音 WAV 文件放置在应用 `Sound\` 目录下，启动时自动加载

## 七、常见问题

### Q: 悬浮窗口不显示？
A: 请确认导航栏右上角「悬浮窗口」按钮为选中状态，或使用 `Ctrl + F9/F10/F11` 快捷键显示。

### Q: 快捷键无效？
A: 快捷键基于 Windows 原生热键机制，如被其他程序占用会弹出提示。可在设置中调整。

### Q: 如何添加自定义提示音？
A: 将 WAV 格式的音频文件放入应用的 `Sound\` 目录，在设置中点击「刷新声音列表」后即可在提示音下拉列表中选择。

### Q: 软件意外关闭后会议数据会丢失吗？
A: 不会。计时状态每 5 秒自动保存，重新启动时可选择恢复未完成的会议。

### Q: 如何批量导出会议记录？
A: 在历史记录页面勾选需要导出的会议，点击「导出」选择保存目录即可。

### Q: 可以保存多个会议计划吗？
A: 可以。点击「保存会议计划」将当前议题保存为模板，通过「加载会议计划」或「管理会议计划」管理已保存的模板。
