# 会帮手 用户操作说明书

## 一、软件简介

会帮手 是一款专业的会议计时工具，帮助用户高效管理会议时间，提高会议效率。软件支持会议计划管理、实时计时、悬浮窗口显示等功能。

## 二、界面概览

### 主界面布局

软件主界面包含以下几个主要区域：

1. **顶部导航栏**：包含 Logo、导航标签和悬浮窗口按钮
2. **主内容区**：根据选中的标签显示不同内容
3. **状态栏**：显示当前时间等信息

### 导航标签

- **会议安排**：创建和管理会议计划
- **历史记录**：查看历史会议记录，支持导出和删除
- **当前会议**：进行中的会议计时界面
- **设置**：软件配置选项

## 三、功能说明

### 3.1 会议安排

在会议安排页面，您可以：

#### 创建会议计划
1. 点击「添加议题」按钮添加议题
2. 为每个议题设置汇报时间和讨论时间
3. 点击「保存会议计划」保存当前计划
4. 点击「开始会议」启动计时器

#### 议题操作
- **添加议题**：在列表末尾添加新议题
- **删除选中**：删除选中的议题（支持多选）
- **上下移动**：调整议题顺序
- **保存/加载计划**：保存会议计划模板或加载已保存的计划

### 3.2 历史记录

在历史记录页面，您可以：

- **查看记录**：浏览所有已结束的会议记录
- **导出数据**：导出选中会议的统计数据为 Excel 文件
- **删除选中**：删除选中的会议记录
- **双击查看详情**：双击会议记录查看详细信息

### 3.3 当前会议

当前会议页面是会议进行时的主要操作界面：

#### 计时器控制
- **开始/暂停**：控制计时器运行状态
- **重置阶段**：重置当前阶段计时
- **下一个阶段**：切换到下一议题或下一阶段
- **上一个阶段**：返回到上一议题或上一阶段

#### 悬浮窗口
- 点击右上角「悬浮窗口」按钮开启/关闭悬浮计时器
- 悬浮窗口始终显示在其他窗口之上，方便在演示时查看

### 3.4 设置

在设置页面，您可以配置以下选项：

#### 议题默认时间
设置新建议题的默认时间：
- **汇报时间**：议题汇报阶段的默认时长
- **讨论时间**：议题讨论阶段的默认时长

#### 悬浮窗口设置
- **透明度**：调整悬浮窗口的透明度
- **窗口大小**：设置悬浮窗口的显示尺寸（小/中/大）

#### 提示设置
配置时间提示相关选项：
- 警告提示选择和时间设置
- 超时提示选择和时间设置

## 四、快捷键说明

软件支持以下快捷键操作：

| 快捷键 | 功能说明 |
|--------|----------|
| `Ctrl + F9` | 打开悬浮窗口（小尺寸） |
| `Ctrl + F10` | 打开悬浮窗口（中尺寸） |
| `Ctrl + F11` | 打开悬浮窗口（大尺寸） |
| `Ctrl + F12` | 显示/隐藏悬浮窗口 |

## 五、使用流程

### 5.1 创建并启动会议

1. **进入会议安排页面**
2. **添加议题**：点击「添加议题」按钮，输入议题名称
3. **设置时间**：为每个议题设置汇报时间和讨论时间
4. **保存计划**（可选）：点击「保存会议计划」保存模板
5. **开始会议**：点击「开始会议」按钮启动计时

### 5.2 进行会议

1. 当前会议页面显示当前议题和剩余时间
2. 点击「下一个阶段」切换到讨论阶段或下一议题
3. 使用悬浮窗口在演示时实时查看时间
4. 会议结束后自动保存到历史记录

### 5.3 查看会议统计

1. 在历史记录页面选中会议记录
2. 双击查看详细统计信息
3. 点击「导出」导出 Excel 报表

## 六、悬浮窗口操作

悬浮窗口提供便捷的时间显示功能：

### 调整大小
使用快捷键 `Ctrl + F9`（小）、`Ctrl + F10`（中）、`Ctrl + F11`（大）调整窗口大小

### 显示/隐藏
使用快捷键 `Ctrl + F12` 或点击主界面「悬浮窗口」按钮控制显示

### 拖拽移动
鼠标拖动悬浮窗口标题栏可移动窗口位置

## 七、常见问题

### Q: 悬浮窗口不显示？
A: 请确保已开启悬浮窗口功能（点击右上角按钮或使用 `Ctrl + F9/F10/F11`）

### Q: 快捷键在 PowerPoint 中无效？
A: 软件使用 `Ctrl + F9-F12` 组合键，避免与 PowerPoint 内置快捷键冲突

### Q: 会议记录如何导出？
A: 在历史记录页面选中会议，点击「导出」按钮即可导出为 Excel 文件
