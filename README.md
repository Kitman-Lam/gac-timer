# 会帮手 用户操作说明书

## 一、软件简介

会帮手（英文名：MeetTimer）是一款专业的会议计时工具，帮助用户高效管理会议时间，提高会议效率。软件支持会议计划管理、Excel批量导入议程、实时计时、悬浮窗口显示等功能。

## 二、界面概览

### 主界面布局

软件主界面包含以下几个主要区域：

1. **顶部导航栏**：包含 Logo、导航标签和悬浮窗口按钮
2. **主内容区**：根据选中的标签显示不同内容
3. **状态栏**：显示当前时间等信息

### 导航标签

- **会议安排**：创建和管理会议计划，支持 Excel 导入
- **历史记录**：查看历史会议记录，支持导出和删除
- **当前会议**：进行中的会议计时界面
- **设置**：软件配置选项

## 三、功能说明

### 3.1 会议安排

在会议安排页面，您可以：

#### 创建会议计划
1. 在「会议名称」输入框中输入会议名称
2. 点击「导入会议计划」链接，可下载模板或导入 Excel 议程文件
3. 点击「添加议题」按钮添加议题
4. 为每个议题设置汇报时间和讨论时间
5. 点击「保存会议计划」保存当前计划
6. 点击「开始会议」启动计时器

#### Excel 导入会议议程
- 点击会议名称输入框右侧的「导入会议计划」文字链接
- 弹出导入对话框，包含三个按钮：
  - **导入文件**：选择 Excel 文件导入（议程将自动填充）
  - **下载模板**：下载标准 Excel 模板
  - **关闭**：取消导入
- Excel 文件名（不含扩展名）将自动作为会议名称
- Excel 模板包含三列：议题、汇报时间（分钟）、讨论时间（分钟）

#### 议题操作
- **添加议题**：在列表末尾添加新议题
- **删除选中**：删除选中的议题（支持多选）
- **上下移动**：调整议题顺序
- **保存/加载计划**：保存会议计划模板或加载已保存的计划
- **管理会议计划**：编辑、修改名称、删除已保存的计划

#### 会议计划管理
- **编辑计划**：加载已保存的计划进行编辑
- **修改名称**：修改计划的名称
- **删除选中**：删除不再需要的计划
- **关闭**：关闭管理窗口

### 3.2 历史记录

在历史记录页面，您可以：

- **查看记录**：浏览所有已结束的会议记录
- **导出数据**：导出选中会议的统计数据为 Excel 文件
- **删除选中**：删除选中的会议记录
- **双击查看详情**：双击会议记录查看详细信息

### 3.3 当前会议

当前会议页面是会议进行时的主要操作界面：

#### 计时器控制
- **开始/暂停**：控制计时器运行状态
- **重置阶段**：重置当前阶段计时
- **下一个阶段**：切换到下一议题或下一阶段
- **上一个阶段**：返回到上一议题或上一阶段
- **结束会议**：手动结束当前会议

#### 临时添加议题
在会议进行中，可临时添加新议题：
- 点击「临时添加议题」按钮
- 输入议题名称、汇报时间、讨论时间
- 选择插入位置（当前议题之后或末位）

#### 悬浮窗口
- 点击右上角「悬浮窗口」按钮开启/关闭悬浮计时器
- 悬浮窗口默认显示在屏幕顶部居中位置
- 悬浮窗口始终显示在其他窗口之上，方便在演示时查看

### 3.4 设置

在设置页面，您可以配置以下选项：

#### 议题默认时间
设置新建议题的默认时间：
- **汇报时间**：议题汇报阶段的默认时长
- **讨论时间**：议题讨论阶段的默认时长

#### 悬浮窗口设置
- **透明度**：调整悬浮窗口的透明度
- **窗口大小**：设置悬浮窗口的显示尺寸（小/中/大）

#### 提示设置
配置时间提示相关选项：
- 警告提示选择和时间设置
- 超时提示选择和时间设置

## 四、快捷键说明

软件支持以下快捷键操作：

| 快捷键 | 功能说明 |
|--------|----------|
| `Ctrl + F9` | 打开悬浮窗口（小尺寸） |
| `Ctrl + F10` | 打开悬浮窗口（中尺寸） |
| `Ctrl + F11` | 打开悬浮窗口（大尺寸） |
| `Ctrl + F12` | 显示/隐藏悬浮窗口 |

## 五、使用流程

### 5.1 创建并启动会议（手动添加议题）

1. **进入会议安排页面**
2. **输入会议名称**：在会议名称输入框中输入名称
3. **添加议题**：点击「添加议题」按钮，输入议题名称
4. **设置时间**：为每个议题设置汇报时间和讨论时间
5. **保存计划**（可选）：点击「保存会议计划」保存模板
6. **开始会议**：点击「开始会议」按钮启动计时

### 5.2 创建并启动会议（Excel 导入）

1. **下载模板**：点击「导入会议计划」链接 → 「下载模板」
2. **填写模板**：在 Excel 中填写议题、汇报时间、讨论时间
3. **保存文件**：将 Excel 文件以会议名称命名保存
4. **导入文件**：点击「导入会议计划」链接 → 「导入文件」选择文件
5. **确认导入**：如有现有数据，确认覆盖
6. **开始会议**：点击「开始会议」按钮启动计时

### 5.3 进行会议

1. 当前会议页面显示当前议题和剩余时间
2. 点击「下一个阶段」切换到讨论阶段或下一议题
3. 使用悬浮窗口在演示时实时查看时间
4. 会议结束后自动保存到历史记录

### 5.4 查看会议统计

1. 在历史记录页面选中会议记录
2. 双击查看详细统计信息
3. 点击「导出」导出 Excel 报表

## 六、悬浮窗口操作

悬浮窗口提供便捷的时间显示功能：

### 调整大小
使用快捷键 `Ctrl + F9`（小）、`Ctrl + F10`（中）、`Ctrl + F11`（大）调整窗口大小

### 显示/隐藏
使用快捷键 `Ctrl + F12` 或点击主界面「悬浮窗口」按钮控制显示

### 拖拽移动
鼠标拖动悬浮窗口标题栏可移动窗口位置

## 七、常见问题

### Q: 悬浮窗口不显示？
A: 请确保已开启悬浮窗口功能（点击右上角按钮或使用 `Ctrl + F9/F10/F11`）

### Q: 快捷键在 PowerPoint 中无效？
A: 软件使用 `Ctrl + F9-F12` 组合键，避免与 PowerPoint 内置快捷键冲突

### Q: 会议记录如何导出？
A: 在历史记录页面选中会议，点击「导出」按钮即可导出为 Excel 文件

### Q: 如何批量导入会议议程？
A: 点击「导入会议计划」链接，下载 Excel 模板，填写后导入即可。Excel 文件名将作为会议名称。

### Q: 可以保存多个会议计划吗？
A: 可以。点击「保存会议计划」可保存当前计划为模板，通过「管理会议计划」可查看和管理所有已保存的计划。
